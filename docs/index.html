Introduction to the RL78 development tools

This lab aims to introduce the Renesas RL78 development tools and provide guidance on the installation of e2studio for RL78.
Once you have successfully read this page and are confident you fulfill the requirements simply click the Get Started! button.

The following are a few important notes for navigating this lab.

  • The reader should combine the written instructions with the animations (gif's) to follow/understand the steps.
  • All images can be opened in a new window by clicking on them.
  • All images can be hovered over to increase their size.
  • At any point this home page can be reached by using the navigation menu on the left side of the browser window.

  • My Renesas Account
