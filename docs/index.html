Introduction to the RL78/G23 development tools

This lab aims to introduce the Renesas RL78/G23 development tools and provide guidance on:

  1. Installation of e2studio for RL78.
  2. First time launch of e2studio.
  3. Creating an RL78 project.
  4. Configuring (using the Smart Configurator), building & debugging an RL78 project from scratch.
Once you have fully read & understand this homepage and are confident you fulfill any mandatory requirements outlined below simply click the Get Started! button.

The following are a few important notes for navigating this lab.

  • The reader should combine the written instructions with the animations (gif's) to follow/understand the steps.
  • All images can be opened in a new window by clicking on them.
  • All images can be hovered over to increase their size.
  • At any point this home page can be reached by using the navigation menu on the left side of the browser window.
  • Steps titled [optional] can be skipped without consequence.
  • Steps titled [recommended] can be skipped but the consequence is typically less value or ease in executing the lab steps but where applicable a workaround will be provided.
  • Steps titled [informative] Have no actionable steps but are typically found to be important or useful in understanding the concepts the lab is trying to teach.

The following are the requirements of the lab - without first completing or acquiring the following list items - the lab will not be achievable.
