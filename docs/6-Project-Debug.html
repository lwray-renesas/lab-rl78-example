6. Debugging a Project

Here we will guide you through debugging a project for the RL78.

6.1 - Debug Configuration

What is a debug configuration - it manages peoples debug settings, binaries to flash, debugger to connect to, supply voltage info, security ID's, connection method, data speeds etc.

6.2 - Launching Debug Session

To start one - right click etc. or ensure the project is selected and use the debug icon.

6.3 - Run & Step

The run button needs pressing - running to main can be controlled in the Debug configurations (in 6.1) but outside of the scope of this lab, see user manual for more info.

6.4 - Watch Window

How to present and use the watch window.

6.5 - Real-time Refresh

How to use real time refresh, fastes we can go is 100ms - important note this has a massive impact on cpu performance as each memory read, the CPU is halted. So debugging this way is not indicative of real world performance and release debuggin should be performed - using GPIO's or UART etc which will not halt the CPU.
